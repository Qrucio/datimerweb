# altimer

A comprehensive productivity timer application designed specifically for students and productivity experts, combining proven focus techniques with collaborative features, gamification, and multimedia support to enhance learning and work efficiency.

## Features

### Core Timer Functionality
- **Pomodoro Technique**: Customizable focus sessions with configurable work/break durations
- **Multiple Timer Modes**: Focus, short break, and long break sessions
- **Animated Timer Display**: Smooth digit transitions with visual feedback
- **Session Statistics**: Track daily focus time, break time, and session counts
- **Streak Tracking**: Monitor consecutive days of productivity

### Social Features
- **Friend System**: Connect with classmates and study partners to track each other's productivity status in real-time
- **Live Status Indicators**: See when friends are in focus mode, on break, or idle - perfect for coordinating study sessions
- **Profile System**: Customizable profiles with handles and productivity stats to build accountability networks
- **Server Support**: Create and join productivity servers for group study sessions and collaborative work environments
- **Video Calls**: Integrated video conferencing designed for collaborative learning - keep each other accountable during study sessions, discuss difficult concepts together, work through problems in real-time, and maintain focus through visible presence and peer support

### Gamification
- **Coin System**: Earn coins for completed focus sessions
- **Wallet & Inventory**: Virtual economy with collectible items
- **Achievement System**: Unlock features and rewards through consistent usage
- **Pro Features**: Premium functionality for enhanced experience

### Productivity Tools
- **Smart Notes**: Advanced note-taking system with task management perfect for organizing lecture notes and research
- **Calendar Integration**: Visual calendar with task scheduling to keep track of assignment deadlines and study sessions
- **Task Reminders**: Customizable notification system for staying on top of academic deadlines and work commitments
- **Intention Setting**: Define specific learning objectives or work goals for each focus session to maximize productivity
- **Smart Messages**: AI-powered motivational prompts and productivity insights tailored to your work patterns

### Multimedia Support
- **Music Integration**: Built-in focus music with ambient sound options
- **Lofi Player**: Embedded YouTube lofi streams for background music
- **Video Support**: Picture-in-picture video capabilities
- **Custom Backgrounds**: Personalizable visual themes and backgrounds

### Games & Entertainment
- **Arcade Games**: Built-in games (Snake, Typing Game) for break periods
- **Non-Distracting Design**: Games designed to provide relaxation without addiction
- **Break-time Activities**: Quick entertainment options during scheduled breaks

### Advanced Features
- **Strict Mode**: Browser extension integration for blocking distractions
- **Focus Mode**: Cinematic interface that minimizes UI elements
- **Command Palette**: Quick keyboard navigation and actions
- **Cross-Device Sync**: Real-time synchronization across devices
- **Offline Support**: Local storage for uninterrupted usage

## Usage

### Getting Started
1. Open the application in your browser
2. Complete the onboarding flow to set up your profile
3. Configure your preferred timer settings (focus duration, break length, etc.)
4. Start your first focus session

### Basic Timer Operation
- **Start/Pause**: Use spacebar or click the play button
- **Mode Switching**: Press Tab to cycle between focus and break modes
- **Session Reset**: Click the reset button or use keyboard shortcuts
- **Settings Access**: Press 'S' to open settings modal

### Social Features
- **Add Friends**: Use the social modal to search and add friends by handle
- **View Status**: Monitor friends' productivity through the friends dock
- **Join Servers**: Create or join productivity servers for group work
- **Video Calls**: Start video sessions directly from the social interface

### Productivity Tools
- **Create Tasks**: Use the smart note editor to create scheduled tasks
- **Set Reminders**: Configure custom reminder times for important tasks
- **Track Progress**: View detailed statistics and calendar history
- **Manage Notes**: Organize thoughts with color-coded sticky notes

### Keyboard Shortcuts
- `Space`: Start/pause timer
- `Tab`: Switch between timer modes (when paused)
- `S`: Open settings
- `P`: Play/pause music
- `N`: Open note library
- `Shift + ?`: Show keyboard help
- `Escape`: Close modals and dialogs

## Browser Extension

The application supports a companion browser extension for enhanced strict mode functionality:
- **Chrome**: Available from Chrome Web Store
- **Firefox**: Available from Firefox Add-ons
- **Features**: Site blocking, session tracking, and productivity enforcement
